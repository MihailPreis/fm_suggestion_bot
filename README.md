# Suggestion Bot

Channel suggestions bot for Telegram

# Deploy

1. Create `.env` file with:
    ```
    TELOXIDE_TOKEN=<telegram bot token>
    CHANNEL_ID=<telegram channel id>
    ADMINS_CHAT_ID=<telegram suggestion chat id>
    ```
2. `cargo build`
3. `cargo run`
